# obsolete

- Word: obsolete
- Similar: ostensible

## adjective

- Meaning: no longer used because sth new has been invented out of date
- Chinese: 淘汰的；废弃的；过时的
- Tags: obsolete
- Eg.: obsolete technology 过时技术
- Eg.: With technological changes many traditional skills have become obsolete. 随着技术的革新，许多传统技艺已被淘汰。
