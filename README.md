# EnglishDictionary

English Dictionary

## Dependency

```http request
https://github.com/lsieun/lsieun-utils
```

## Before Run

修改`src/main/resources/config.properties`文件的路径信息：

```txt
english.dictionary.filepath=/path/to/directory/EnglishDictionary
```
