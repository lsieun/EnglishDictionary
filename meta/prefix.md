# prefix

- be-: behead, belabor（be+一个熟悉的单词）
- contra/contro: 相反的 controversial, contradict, controversy
- heter: heterogeneous, 表示“异”；“hetero-”的变体
- homo: homogenous 同源
- para: parallel, parable, paranormal
- per: persevere, peruse
- peri: 周围、周边、边缘 perimeter
- un: unforeseen
- ob
- pre
    - prejudge/prejudice
- tri: triple
